# AI Helper - Обновление 2026-02-19

## Новые функции

### 1. Мультиязычный интерфейс (RU/EN)
- Добавлен модуль `src/localization.py` с системой локализации
- Кнопка переключения языка RU|EN в заголовке окна
- Все тексты интерфейса переведены на английский
- Язык сохраняется в настройках и восстанавливается при запуске
- Меню трея и всплывающие подсказки также локализованы

### 2. Встроенный браузер для ссылок
- Новый модуль `src/overlay/web_dialog.py`
- Ссылки в чате открываются во встроенном браузере
- Навигация: назад/вперёд/обновить
- Кнопка "Открыть в браузере" для внешнего просмотра
- Прогресс-бар загрузки страницы

### 3. Диалог настроек
- Новый модуль `src/overlay/settings_dialog.py`
- API ключ с возможностью показать/скрыть
- Управление списком моделей (добавление/удаление)
- Настройка горячих клавиш
- Чекбокс автозапуска с Windows

### 4. Защита от повторного запуска
- Windows Mutex предотвращает запуск нескольких копий
- Программа тихо завершается если уже запущена

## Исправления

### Локализация трея
- **Проблема**: Меню трея не обновлялось при смене языка
- **Причина**: Разные пути импорта (`from localization` vs `from src.localization`) создавали два независимых экземпляра класса Localization
- **Решение**: Унифицированы все импорты на `from src.localization import ...`

### Горячие клавиши
- Горячие клавиши теперь применяются сразу без перезапуска
- Настройки сохраняются в settings.json

### Детектор контекста
- Игнорирует окна самого AI Helper при определении активного окна
