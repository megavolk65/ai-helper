# AI Helper

Универсальный AI-ассистент с оверлеем на базе YandexGPT. Вызывается в любом приложении горячей клавишей. Основной фокус — помощь в играх, но может отвечать на любые вопросы.

## Цели

* Универсальный помощник для любого приложения (игры, браузер, IDE и т.д.)
* Вызов по горячей клавише или через Stream Deck
* Текстовый чат с ИИ (без голоса)
* Опциональный анализ скриншотов
* Авто-определение контекста (игра/приложение) для релевантных ответов
* Безопасность для античитов
* Бесплатное использование (в рамках лимитов YandexGPT Lite)

## Технический стек

* **Python 3.11+** — основной язык
* **PyQt6** — GUI с always-on-top оверлеем
* **keyboard** — глобальные горячие клавиши
* **pywin32** — определение активного окна (название игры)
* **yandex-cloud-ml-sdk** — официальный SDK для YandexGPT
* **mss** — быстрые скриншоты
* **Pillow** — обработка изображений
* **PyInstaller** — сборка в .exe
* **Inno Setup** — создание инсталятора

## Функциональность

### Горячие клавиши

* `Ctrl+Shift+G` — открыть/скрыть оверлей чата
* `Ctrl+Shift+S` — сделать скриншот и прикрепить к сообщению
* `Escape` — закрыть оверлей

### Оверлей

* Полупрозрачное тёмное окно поверх всех приложений
* Текстовое поле для ввода вопроса
* История сообщений (чат)
* Индикатор текущей игры (определяется автоматически)
* Возможность указать игру вручную
* **Отображение картинок из интернета** — карты локаций, скриншоты, гайды

### Системный трей

* Иконка в трее при запуске
* Контекстное меню: показать/настройки/выход
* Автозагрузка при старте Windows

### Интеграция с YandexGPT

* Динамический системный промпт с контекстом (если игра — добавляет её название)
* Отвечает на любые вопросы, не только по играм
* Поддержка контекста беседы (история сообщений)
* Поиск в интернете для актуальной информации

### Анализ скриншотов

* По отдельной клавише делается скриншот
* Изображение отправляется в Yandex Vision
* Распознанный контекст добавляется к вопросу

## Безопасность (античиты)

Приложение безопасно для использования с любыми играми:

* Используется стандартное Windows-окно (WS_EX_TOPMOST)
* Нет инъекций в процессы игр
* Нет чтения памяти игр
* Нет хуков DirectX/Vulkan
* Определение игры через публичный Win32 API (GetWindowText)
* Скриншоты через стандартный Windows API

Аналогично работают Discord Overlay, GeForce Experience, Xbox Game Bar.

## Этапы разработки

### Этап 1: Базовая структура

* Создать проект и структуру папок
* Настроить виртуальное окружение
* Установить зависимости
* Создать конфигурационный файл

### Этап 2: Определение контекста

* Реализовать получение активного окна
* Извлечение названия приложения/игры из заголовка окна
* Маппинг известных процессов на понятные названия
* Определение типа: игра / браузер / IDE / другое

### Этап 3: Интеграция YandexGPT

* Подключить yandex-cloud-ml-sdk
* Реализовать отправку сообщений
* Настроить системный промпт с контекстом игры
* Обработка истории диалога

### Этап 4: GUI оверлей

* Создать главное окно PyQt6
* Реализовать always-on-top поведение
* Добавить поле ввода и область чата
* Применить тёмную тему с полупрозрачностью
* Парсинг ссылок на изображения в ответах
* Асинхронная загрузка и отображение картинок inline

### Этап 5: Горячие клавиши

* Настроить глобальные хоткеи
* Показ/скрытие оверлея
* Интеграция со Stream Deck (через стандартные хоткеи)

### Этап 6: Скриншоты и Vision

* Реализовать захват экрана
* Интеграция с Yandex Vision API
* Добавление контекста скриншота к запросу

### Этап 7: Автозагрузка и трей

* Сворачивание в системный трей
* Контекстное меню трея (показать/настройки/выход)
* Добавление в автозагрузку Windows (реестр или ярлык в Startup)
* Опция вкл/выкл автозапуска в настройках

### Этап 8: Сборка и инсталятор

* Сборка в .exe через PyInstaller
* Создание инсталятора через Inno Setup
* Иконка на рабочем столе и в меню Пуск
* Деинсталятор

### Этап 9: Финализация

* Тестирование с разными играми
* Тест установки на чистую систему
* Документация (README)

## Требования к пользователю
