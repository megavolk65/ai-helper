## Возможности

- 🎮 **Универсальный помощник** — работает в любом приложении (игры, браузер, IDE)
- 🧠 **Любые модели** — работает с любыми LLM, совместимыми с OpenAI API (GPT, Claude, Gemini, Qwen и др.)
- 🌐 **Встроенный браузер** — открывайте ссылки прямо в оверлее. Больше никакой беготни через **Alt-Tab**!
- ⌨️ **Горячие клавиши** — быстрый вызов из любой точки системы
- 📷 **Зрительный контакт** — AIgator «видит» ваш экран (если прикреплен скриншот), что позволяет задавать максимально короткие вопросы
- 🎯 **Умный контекст** — AIgator автоматически понимает, в какой игре или программе вы находитесь
- 🔒 **Безопасность** — не вызывает срабатывания античитов
- 🌍 **Две версии** — международная и русская (работает без VPN и зарубежных карт)

## Использование

1. Нажмите горячую клавишу чтобы вызвать оверлей
2. Введите вопрос и нажмите Enter
3. Получите ответ от AI

### Умный контекст и жизнь без Alt-Tab
AIgator создан для того, чтобы вы не тратили время на лишние объяснения и переключения:
- **Забудьте про Alt-Tab**: Получайте ответы и просматривайте веб-ресурсы, не выходя из игры или приложения.
- **Визуальный контекст**: Прикрепите скриншот, и AIgator мгновенно увидит ситуацию вашими глазами. Вы можете просто спросить «Что это за предмет?» или «Как пройти этот квест?».
- **Качество ответов и галлюцинации**: Помните, что качество помощи зависит от выбранной модели. ИИ иногда может «галлюцинировать» (давать недостоверную информацию).
- **Проверка через ссылки**: Вы всегда можете попросить AIgator прислать ссылки на гайды, вики или карты и открыть их во **встроенном браузере**, чтобы перепроверить информацию или изучить детали.

## Безопасность для античитов

Приложение **безопасно** для использования с любыми играми:

- ✅ Использует стандартное Windows-окно (как Discord, Steam)
- ✅ Не инжектится в процессы игр
- ✅ Не читает память игр
- ✅ Не использует хуки DirectX/Vulkan
- ✅ Скриншоты через стандартный Windows API

### Важное ограничение: Режим экрана
Для корректной работы оверлея игра должна быть запущена в режиме **«Окно без рамок»** (Borderless Window) или **«В окне»**. В режиме **«Эксклюзивный полный экран»** (Exclusive Fullscreen) оверлей не сможет отобразиться поверх игры.


## Установка

### Готовый установщик

Скачайте `AIgator_Setup_x.x.x.exe` из [Releases](https://github.com/megavolk65/AIgator/releases)

### Сборка из исходников

```bash
git clone https://github.com/megavolk65/AIgator.git
cd AIgator
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python main.py
```

### Создание EXE и установщика

```bash
pip install pyinstaller
pyinstaller build.spec
# Затем скомпилировать installer/setup.iss через Inno Setup
```
