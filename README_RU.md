## Возможности

- 🎮 **Универсальный помощник** — работает в любом приложении (игры, браузер, IDE).
- 🌐 **Встроенный браузер** — открывайте ссылки прямо в оверлее. Больше никакой беготни через **Alt-Tab**!
- 🧠 **Умный контекст** — AIgator автоматически понимает, в какой игре или программе вы находитесь.
- 📷 **Зрительный контакт** — AIgator «видит» ваш экран (если прикреплен скриншот), что позволяет задавать максимально короткие вопросы: «Как пройти это место?» или «Что это за предмет?».
- ⌨️ **Горячие клавиши** — быстрый вызов из любой точки системы (`PageUp` / `PageDown`).
- 🔒 **Безопасность** — не вызывает срабатывания античитов, не инжектится в процессы.
- 🌍 **Две версии** — международная и русская (работает без VPN и зарубежных карт).
- ⚠️ **Качество ответов** — Помните, что качество помощи зависит от выбранной модели. ИИ иногда может «галлюцинировать». Вы всегда можете перепроверить инфо через ссылки во встроенном браузере.

## Быстрый старт

1. **Скачайте** и запустите приложение.
2. **Откройте настройки** (⚙️) и выберите API провайдер.
3. **Введите API ключ** и добавьте нужные модели.
4. **Готово!** Нажмите `PageUp` для вызова оверлея или `PageDown` для скриншота.

### API провайдеры

#### OpenRouter (Международный)
- 🌐 **Сайт:** [openrouter.ai](https://openrouter.ai)
- 🔑 **Ключ:** [openrouter.ai/keys](https://openrouter.ai/keys)
- ✅ **Бесплатные модели:** Модели с суффиксом `:free` работают без оплаты.

#### AITunnel (Для РФ)
- 🌐 **Сайт:** [aitunnel.ru](https://aitunnel.ru)
- 🔑 **Ключ:** В личном кабинете после регистрации.
- 💳 **Оплата:** Российские карты, рубли.
- ⚠️ **Бесплатных моделей нет.**

## Безопасность и Режим экрана

Приложение **безопасно** для использования с любыми играми:
- ✅ Использует стандартное Windows-окно (как Discord, Steam).
- ✅ Не инжектится в процессы и не читает память игр.
- ✅ Скриншоты через стандартный Windows API.

### ⚠️ Важное ограничение: Режим экрана
Для корректной работы оверлея игра должна быть запущена в режиме **«Окно без рамок»** (Borderless Window) или **«В окне»**. В режиме **«Эксклюзивный полный экран»** (Exclusive Fullscreen) оверлей не сможет отобразиться поверх игры.

## Установка

### Готовый установщик
Скачайте `AIgator_Setup_x.x.x.exe` из [Releases](https://github.com/megavolk65/AIgator/releases).

### Сборка из исходников
```bash
git clone https://github.com/megavolk65/AIgator.git
cd AIgator
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python main.py
```
